Console program that prints an image to console using specified chars.

For project building: javac -d target/ src/java/edu/school21/printer/*/*.java

For project run:  java -classpath target/ edu.school21.printer.app.Main --path="path" --white="char" --black="char"
Where: --path="image path"
       --white="char for white color replacing"
       --black="char for black color replacing"
